# Skills Manager 使用演示

## 功能展示

### 1. 浏览所有技能

打开界面后，你会看到所有可用的技能卡片，每个卡片显示：
- 技能名称
- 技能类别
- 详细描述
- 安装状态（已安装/可用）

### 2. 搜索技能

在搜索框中输入关键词，例如：
- "test" - 查找所有测试相关的技能
- "documentation" - 查找文档相关的技能
- "python" - 查找 Python 相关的技能

### 3. 按类别筛选

点击顶部的类别标签：
- **All** - 显示所有技能
- **Code Generation** - 代码生成类技能
- **Testing** - 测试类技能
- **Documentation** - 文档类技能
- **Code Quality** - 代码质量类技能
- **Requirements** - 需求类技能
- **DevOps** - 开发运维类技能
- **Debugging** - 调试类技能
- **Verification** - 验证类技能
- **Maintenance** - 维护类技能

### 4. 选择技能

有两种方式选择技能：
1. 点击技能卡片（整个卡片都可点击）
2. 直接勾选卡片右上角的复选框

选中的技能会：
- 卡片变为绿色背景
- 复选框被勾选
- 底部显示选中数量

### 5. 安装技能

#### 方式一：安装所有技能

1. 点击 **"📦 Install All Skills"** 按钮
2. 弹出确认对话框，显示将要安装的技能数量
3. 点击确认
4. 等待安装完成，右上角会显示成功通知

#### 方式二：安装选定的技能

1. 选择你需要的技能（可以多选）
2. 点击 **"✅ Install Selected Skills"** 按钮
3. 弹出确认对话框
4. 点击确认
5. 等待安装完成

### 6. 查看安装状态

- **已安装的技能**：卡片显示灰色背景，标记为 "✓ Installed"
- **可用的技能**：卡片显示白色背景，标记为 "○ Available"
- 已安装的技能无法再次选择（复选框被禁用）

### 7. 刷新列表

点击 **"🔄 Refresh"** 按钮可以刷新技能列表，查看最新的安装状态。

## 使用场景示例

### 场景 1：新用户首次使用

1. 启动 Skills Manager
2. 浏览所有可用的技能
3. 点击 "Install All Skills" 一键安装所有技能
4. 等待安装完成
5. 打开 Claude Code，所有技能已可用

### 场景 2：只安装测试相关的技能

1. 启动 Skills Manager
2. 点击 "Testing" 类别标签
3. 查看所有测试相关的技能
4. 点击 "Install All Skills" 或选择特定技能
5. 安装完成

### 场景 3：搜索并安装特定技能

1. 启动 Skills Manager
2. 在搜索框输入 "python test"
3. 查看搜索结果
4. 选择需要的技能（如 Python Test Updater）
5. 点击 "Install Selected Skills"
6. 安装完成

### 场景 4：按需安装

1. 启动 Skills Manager
2. 浏览技能列表
3. 阅读每个技能的描述
4. 选择符合当前项目需求的技能
5. 批量安装选定的技能

## 安装后验证

安装完成后，可以通过以下方式验证：

### 方法 1：使用 Claude Code CLI

```bash
claude skills list
```

应该能看到已安装的技能列表。

### 方法 2：检查文件系统

```bash
ls ~/.claude/skills/
```

应该能看到已安装的技能目录。

### 方法 3：在 Claude Code 中使用

打开 Claude Code，尝试使用已安装的技能：

```
Use the skill 'unit-test-generator' to generate tests for my code
```

## 提示和技巧

1. **批量安装**：如果你是新用户，建议先安装所有技能，然后根据实际使用情况决定保留哪些

2. **按需安装**：如果你清楚自己的需求，可以只安装相关类别的技能，节省磁盘空间

3. **定期更新**：当仓库更新后，重新运行 Skills Manager 可以安装新增的技能

4. **搜索功能**：不确定技能名称时，使用搜索功能可以快速找到相关技能

5. **类别筛选**：按类别筛选可以帮助你系统地了解每个领域的可用技能

## 故障排除

### 安装失败

如果安装失败，检查：
1. `~/.claude/skills/` 目录是否存在且可写
2. 磁盘空间是否充足
3. 后端服务是否正常运行

### 技能不显示

如果技能列表为空：
1. 确认后端服务正在运行
2. 检查浏览器控制台是否有错误
3. 尝试点击 "Refresh" 按钮

### Claude Code 看不到技能

如果安装后 Claude Code 看不到技能：
1. 重启 Claude Code 应用
2. 运行 `claude skills list` 验证安装
3. 检查技能目录权限

## 技术细节

### 安装过程

1. 前端发送安装请求到后端 API
2. 后端从仓库复制技能目录到 `~/.claude/skills/`
3. 如果技能已存在，会先删除旧版本再安装新版本
4. 安装完成后返回结果给前端
5. 前端显示成功通知并刷新列表

### 数据流

```
用户操作 → 前端 JavaScript → HTTP 请求 → Flask 后端 → 文件系统操作 → 响应 → 前端更新
```

### 安全性

- 所有操作都在本地进行，不涉及网络传输
- 只能安装仓库中已有的技能
- 使用标准的文件系统权限控制
